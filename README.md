# DRIFTY (Vite + modules)

## Ontwikkelen
```
npm install
npm run dev
```

Maak eerst een `.env` (of kopieer `.env.example`) met je Supabase project:

```
VITE_SUPABASE_URL="https://minpgkoqkrqifkfwhshp.supabase.co"
VITE_SUPABASE_ANON_KEY="<vul-hier-je-anon-key-in>"
```

## Wat zit erin?
- Vite bundler
- Modulaire UI (auth, home/kaart, reserveren, profiel, admin, leaderboard)
- LocalStorage state
